Sobre mim

Meu nome é Elen Carvalho

Sou formada em Engenharia Química, tenho formação técnica em química e mineração. Em minha carreira profisional adquiri experiências com análises físico químicas, controle de qualidade e visão de processos nas indústrias farmacêutica, petroquímica e alimentícia.

Atualmente, dedico aos estudos de Análise de Dados e Ciências de Dados, para adquirir experiência na solução de problemas de negócio e domínio sobre as ferramentas de análise de dados.

Estou buscando uma oportunidade de trabalhar profissionalmente como Cientista de Dados para melhorar a tomada de decisão da empresa, através da construção de soluções usando dados. .

    4+ Projetos completos de Ciência de Dados

    Construção de soluções de dados para problemas de negócio, próximos dos desafios reais das empresas, utilizando dados públicos de competições de Ciência de Dados, onde eu abordei o problema desde a concepção do desafio de negócio até a publicação do algoritmo treinado em produção, utilizando ferramentas de Cloud Computing.

    1 ano como Analista de Implementação

    Atuando na implementação de sistema Lins, software especializado para laboratórios de análises ambiental e alimentos em conformidade com as diretrizes da NBR ISO 17025. Contato com o cliente, validação e manipulação dos dados a serem implementados e a inserção desses dados no sistema utilizando ferramentas como MySQL, Excel e Visual Studio Code.

    9+ anos como Técnica em Química

    Atuando com análises laboratóriais, interpletação de resultados e controle de qualidade seguindo ISOS e Legislação pertinenetes para área. Com essa experiência desenvolvi muito bem a visão da rotina indústrial desde a matéria prima a ser trabalhada até o bem de consumo finalizado.

    2+ anos como Analista Comercial

    Atuação no setor Técnico Comercial voltado para Monitormanentos Ambientais. Nessa experiência desenvolvi a comunicação com o cliente e habilidades técnicas para para o levantamento do monitoramento seguindo as normas de estabelecidas pelos órgãos ambientais e negociação comercial.

    Desenvolvimento de um Painel Gerencial para Negócio com o Streamlit.

    Desenvolvimento de um Painel Gerencial para Negócio com o Streamlit.

    Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimento web como o Streamlit e Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de restaurantes.
